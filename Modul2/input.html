UJI COBA INPUT HTML

Uji coba 1




Kesimpulan: Dengan atribut "name" yang sama pada semua radio button, pengguna hanya dapat memilih satu opsi karena termasuk dalam grup yang sama. Jika "name" berbeda, setiap radio button dapat dipilih secara bersamaan karena dianggap sebagai grup terpisah.

Uji coba 2




Kesimpulan: Checkbox dengan "name" yang sama memungkinkan pengguna memilih lebih dari satu opsi karena data akan dikirim sebagai array. Jika "name" berbeda, setiap checkbox dianggap sebagai field terpisah.

Uji coba 3




Kesimpulan: Type="color" menampilkan color picker, type="date" menampilkan kalender, dan type="button" menampilkan tombol yang dapat diprogram dengan JavaScript.

Uji coba 4




Kesimpulan: Type datetime-local menampilkan input tanggal dan waktu, type month menampilkan bulan dan tahun, type email melakukan validasi format email otomatis. Tampilan dapat berbeda antar browser.

Uji coba 5




Kesimpulan: Type="image" berfungsi sebagai tombol submit dengan gambar, type="file" membuka dialog pemilihan file, type="hidden" menyimpan data yang tidak terlihat oleh pengguna.

Uji coba 6



Kesimpulan: Type="number" membatasi input hanya angka dengan tombol increment/decrement. Type="range" menampilkan slider. Jika min diubah menjadi -50, input number menerima nilai negatif tergantung browser, range akan menyesuaikan rentang slider.

Uji coba 7




Kesimpulan: Type="url" memvalidasi format URL, type="search" menyediakan kotak pencarian dengan tombol hapus, type="tel" tidak melakukan validasi otomatis namun dapat dikombinasikan dengan atribut pattern untuk validasi format.

Uji coba 8



Kesimpulan: Type="time" menampilkan input waktu (jam dan menit), type="week" menampilkan minggu dalam setahun. Format tampilan dapat bervariasi antar browser.
