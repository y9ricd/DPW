FORM SEDERHANA





Jika Anda mengklik tombol "Kirim", data formulir akan dikirim ke halaman bernama "/action_page.php".

Kesimpulan Ujicoba:
- Tombol Reset mengembalikan form ke nilai awal (Farid ganteng)
- Tombol Kirim mengirim data ke action_page.php
- Atribut target="_self" membuka hasil di tab yang sama, "_blank" membuka di tab baru
- Method="get" mengirim data melalui URL (terlihat di address bar), method="post" mengirim melalui body HTTP (tidak terlihat)
- GET cocok untuk request data, POST untuk mengirim data sensitif atau dalam jumlah besar
