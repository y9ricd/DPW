8. Penjelasan AJAX dan jQuery

Apa itu AJAX?

AJAX (Asynchronous JavaScript and XML) adalah cara mengirim dan menerima data dari server tanpa perlu me-reload halaman. Contoh: memuat komentar baru, mengirim form, atau mengambil data dari database secara langsung.

Apa itu jQuery?

jQuery adalah library JavaScript yang memudahkan manipulasi elemen HTML, menangani event, membuat animasi, dan menggunakan AJAX dengan kode yang lebih singkat.

Cara Menyertakan jQuery

<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.7.1/jquery.min.js"></script>

Contoh Penggunaan jQuery Sederhana

// Sembunyikan / tampilkan elemen saat tombol diklik
$("#tombol").click(function() {
    $("#kotak").slideToggle("slow");
});
